DISCREPANCIES FOUND:
1. ⚠️ Capital % selector missing 50% and 100% options (only shows 5%-25%)
2. ⚠️ Date column for past days: Spec says "age/duration open" right-aligned. Day 3 (gift) shows no age — correct since gift days are auto-completed. But should gift days show any age indicator?
3. ⚠️ Rating system: Spec mentions ratings 🏆👑 (4, ≥20%) and 🏆🏆👑💰 (5, ≥50%) from brainstorming, but these are NOT in the spec document. The spec only mentions 🏆 and 🏆🏆 as examples. The full rating table from brainstorming is missing from the spec.
4. ⚠️ Spec says "Trade Capital, Target, Proj Capital for today's sub-rows: first row shows values normally, subsequent rows show dimmed." Mockup shows this correctly with dim-value class.
5. ⚠️ The spec doesn't explicitly mention the "new" label shown in the new trade input row's Date column. Minor visual detail.
6. ⚠️ Spec says Actual Capital includes "realized + unrealized" but for today's individual trade rows, mockup shows dashes (—) which is correct since Actual Capital is only meaningful at the day summary level.
