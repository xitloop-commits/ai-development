DHAN SCRIP MASTER CSV FORMAT
============================

COMPACT CSV URL: https://images.dhan.co/api-data/api-scrip-master.csv
DETAILED CSV URL: https://images.dhan.co/api-data/api-scrip-master-detailed.csv
SEGMENT-WISE API: GET https://api.dhan.co/v2/instrument/{exchangeSegment}

COMPACT CSV COLUMNS (what we'll use):
- SEM_EXM_EXCH_ID: Exchange (NSE, BSE, MCX)
- SEM_SEGMENT: Segment (C=Currency, D=Derivatives, E=Equity, M=Commodity)
- SEM_INSTRUMENT_NAME: Instrument type defined by exchange
- SEM_EXPIRY_CODE: Expiry code for futures
- SM_SYMBOL_NAME: Symbol name
- SEM_TRADING_SYMBOL: Exchange trading symbol
- SEM_CUSTOM_SYMBOL: Dhan display symbol
- SEM_EXCH_INSTRUMENT_TYPE: Detailed instrument type
- SEM_SERIES: Exchange series
- SEM_LOT_UNITS: Lot size
- SEM_EXPIRY_DATE: Expiry date (for derivatives)
- SEM_STRIKE_PRICE: Strike price (for options)
- SEM_OPTION_TYPE: CE or PE
- SEM_TICK_SIZE: Minimum price increment
- SEM_EXPIRY_FLAG: M=Monthly, W=Weekly

KEY COLUMNS FOR SECURITY ID LOOKUP:
- SEM_SMST_SECURITY_ID (the security ID we need)
- SEM_EXM_EXCH_ID (exchange filter)
- SM_SYMBOL_NAME (underlying name like NIFTY, BANKNIFTY, CRUDEOIL)
- SEM_EXPIRY_DATE (expiry filter)
- SEM_STRIKE_PRICE (strike filter)
- SEM_OPTION_TYPE (CE/PE filter)
- SEM_INSTRUMENT_NAME (OPTIDX, FUTIDX, OPTCOM, FUTCOM, etc.)

FOR MCX FUTCOM RESOLUTION:
- Filter: SEM_EXM_EXCH_ID = MCX, SEM_INSTRUMENT_NAME = FUTCOM
- Sort by SEM_EXPIRY_DATE ascending
- Pick nearest-month (first non-expired)

NOTE: The compact CSV has the security ID as the first column (unnamed or SEM_SMST_SECURITY_ID)
