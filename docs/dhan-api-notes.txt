DHAN API v2 DOCUMENTATION NOTES
================================

BASE URL: https://api.dhan.co/v2/
AUTH: Header 'access-token: JWT'

RATE LIMITS:
- Order APIs: 10/sec, 250/min, 1000/hr, 7000/day
- Data APIs: 5/sec, 100000/day
- Quote APIs: 1/sec, unlimited/min
- Non Trading APIs: 20/sec, unlimited
- Order modifications: max 25/order

SIDEBAR SECTIONS (all available APIs):
Trading APIs:
  - Orders (place, modify, cancel, slicing, order book, trade book)
  - Super Order
  - Forever Order
  - Conditional Trigger
  - Portfolio and Positions
  - EDIS
  - Trader's Control (Kill Switch)
  - Funds & Margin
  - Statement
  - Postback
  - Live Order Update (WebSocket)

Data APIs:
  - Market Quote
  - Live Market Feed (WebSocket)
  - Full Market Depth
  - Historical Data
  - Expired Options Data
  - Option Chain

Other:
  - Annexure (exchange segments, error codes)
  - Instrument List

ORDER ENDPOINTS:
  POST   /orders                          - Place new order
  PUT    /orders/{order-id}               - Modify pending order
  DELETE /orders/{order-id}               - Cancel pending order
  POST   /orders/slicing                  - Slice over freeze limit
  GET    /orders                          - Order book (all orders today)
  GET    /orders/{order-id}               - Single order status
  GET    /orders/external/{correlation-id} - Order by correlation ID
  GET    /trades                          - Trade book (all trades today)
  GET    /trades/{order-id}               - Trades of an order

ORDER PLACEMENT PARAMS:
  dhanClientId (required) - string
  correlationId - string (max 30 chars)
  transactionType (required) - BUY | SELL
  exchangeSegment (required) - NSE_EQ | NSE_FNO | MCX_COMM | BSE_EQ | etc.
  productType (required) - CNC | INTRADAY | MARGIN | MTF | CO | BO
  orderType (required) - LIMIT | MARKET | STOP_LOSS | STOP_LOSS_MARKET
  validity (required) - DAY | IOC
  securityId - string (exchange standard ID)
  quantity (required) - int
  price (required) - float
  triggerPrice - float (for SL orders)
  afterMarketOrder - boolean
  boProfitValue - float (bracket order target)
  boStopLossValue - float (bracket order SL)

ORDER STATUS VALUES: TRANSIT | PENDING | REJECTED | CANCELLED | TRADED | EXPIRED

NOTE: Order Placement/Modification/Cancellation requires Static IP whitelisting
